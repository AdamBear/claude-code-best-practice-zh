---

**终端**
- 打开 Terminal（按 `Cmd + Space`，输入 "Terminal"，然后回车）

**Homebrew**
- 先检查 Homebrew 是否已经安装：
  ```bash
  brew --version
  ```
- 如果出现 "command not found"，先安装 Homebrew：
  ```bash
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  ```

**Claude Code**
- ```bash
  brew install --cask claude-code
  ```

**验证**
- ```bash
  claude --version
  ```
